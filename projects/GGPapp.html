OBJECTIVE

As one of our products at Controlco, we provide a mobile version of our building automation websites. This version is heavily used by technicians and electricians onsite when they're maintaining data centers and central plant equipment. Controlco's mobile site is incredibly outdated and cluttered so I thought about creating an app version instead, using better design trends. The app would be contain all the numeric data and information that the desktop site hosts, but without the visual graphic representations of the equipment. I felt that there is a need for this design because the current mobile site does not provide the users with the information they need up front -- users would have to open collapsible menus repetitively to seek the information. With an app, users can simply swipe and use gestures to focus on one screen at a time.

A portable way to track HVAC, energy, and security data.



APPROACH

I researched a little bit into what our users like to see when they log in to check their buildings. We typically provide them access to a 2D graphic of their equipment, but because this is mobile version, I decided to remove the excess and keep it minimal. This app would be compatible for both iOS and Android although I created a mock-up using iPhone 7 screen dimensions. After figuring out the architecture of data, I created some wireframes.



A wireframe representation of the user flow. The global navigation appears on the botttom of each page.


For this particular client, we wanted to go with a blue gradient as the background to make it look a bit more captivating than the usual flat colors we've used. Using Avenir Next made the text much more legible and was condensed enough to fit enough characters without looking squished together.



The color palette for this project is minimal and clean.


The Weatherdog mascot. Instead of making a high fidelity icon for our logo, we wanted to keep things traditional and cartoonish instead.


A set of weather icons I created for this website. These icons are used to show the different weather conditions.



FUTURE GOALS

The finished design was handed over to my colleague and the project is currently in its beta phase. We will be ready to implement changes and feedback to keep the site updated, but we will also like to work with third party agencies to incorporate disaster relief efforts. We would like to include resources such as links that will provide donation tips during these disasters.
